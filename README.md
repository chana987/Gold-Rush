# Gold-Rush

Two-player game. Get as many coins as you can to win the round.
Player 1 uses keys w, a, s, d to move.
Player 2 uses keys i, j, k, l.
Adjust board size to change difficulty.

## Installation

```bash
npm i
```

## `node server.js`

Open localhost:3000
